# 🌦️ Weather App

A simple and interactive Weather App built using **Python** and **Tkinter** that provides real-time weather information for any city using the OpenWeatherMap API.

## 📌 Features

- Search weather by city name
- Current temperature
- Humidity
- Weather description
- Wind speed
- Weather icon
- Celsius/Fahrenheit unit selection
- User-friendly GUI
- Input validation
- Error handling for invalid city names
- Network error handling

## 🛠 Technologies Used

- Python
- Tkinter
- Requests
- Pillow (PIL)
- OpenWeatherMap API

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/Nasrin-Code/OIBSIP.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Get a free API key from:

https://openweathermap.org/api

4. Replace

```python
API_KEY = "YOUR_API_KEY"
```

with your own API key.

5. Run the application

```bash
python weather_gui.py
```
